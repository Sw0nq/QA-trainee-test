# Тестовое задание для стажёра QA (Автотесты API)

## Описание

Этот репозиторий содержит решение Задания 2.1: тест-кейсы и автоматизированные тесты для API микросервиса по работе с объявлениями.

Автотесты написаны на Python с использованием фреймворков `pytest` и `requests` для проверки функциональности API согласно предоставленной документации.

## Требования

*   Python 3.8+
*   pip

## Инструкция по запуску

1.  **Склонируйте репозиторий:**
    ```bash
    git clone https://github.com/Sw0nq/QA-trainee-test.git
    cd QA-trainee-test
    ```

2.  **Создайте и активируйте виртуальное окружение (рекомендуется):**
    *   Для Windows:
        ```bash
        python -m venv venv
        .\venv\Scripts\activate
        ```
    *   Для macOS/Linux:
        ```bash
        python3 -m venv venv
        source venv/bin/activate
        ```

3.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Запустите тесты:**
    Выполните команду в корневой папке проекта:
    ```bash
    pytest -v
    ```
    Ключ `-v` (verbose) выведет подробный отчет о прохождении каждого теста. Вы увидите, что все тесты, кроме одного пропущенного, успешно пройдены. Один тест специально помечен как `@pytest.mark.skip`, так как он выявляет известный баг в API (возврат 500 ошибки вместо 400).
